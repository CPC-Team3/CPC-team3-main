# setup enviroment
> ## setup workspace
> 1. clone this reprository to you machine
> 2. open java eclipse
> 3. choose "CPC-team3-main" as work space when booting java

> ## add project to work space
> 1. open: file > import > General > existing project into workspce 
> 2. click next
> 3. in "select root directory" choose "capstone_project" folder (inside "CPC-team3-main" folder)
> 4. select (tick) capstone_project(<oath to the folder>)
> 5. click finish
> 6. close welcome window

> ## setup library
> 1. right click on the project 
> 2. select properties 
> 3. to "java build path" 
> 4. in  "java build path": go to "libraries tab"
> 5. in "libraries tab": dounle click on  "JRE System Library" (under "Modulepath")
> 6. select Alternate JRE
> 7. choose "jre" or other
> 8. click finish
> 9. click apply and close

# run full simulation
> 1. go to capstone_project > src > simulation
> 2. run Simulation.java
